Это научный калькулятор специально для девушек-гиков.
Вычисляет синусы, степени, косинусы, факториалы, корни квадратные, логарифмы.
Исходники нашла на просторах американско-канадского интернета.
Сама писала только CSS-стили и кое-что из HTML. JS не мои.
Все стили и скрипты внутри HTML, подключены с помощью тегов ибо по задумке должен быть один файл единый.
Дизайн мой. Цветовая гамма "вытянута" мной c помощью Adobe Color из картинки с бабиной ниток.
