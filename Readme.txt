Это научный калькулятор специально для девушек-гиков.
Вычисляет синусы, степени, косинусы, факториалы, корни квадратные, логарифмы.
Исходники нашла на просторах американско-канадского интернета.
Сама писала только CSS-стили и кое-что из HTML. JS не мои.
Все стили и скрипты внутри HTML, подключены с помощью тегов ибо по задумке должен быть один файл единый.
Дизайн мой. Цветовая гамма "вытянута" мной c помощью Adobe Color из картинки с бабиной ниток.

Adobe Color находится десь: https://color.adobe.com/create 


This is a scientific calculator especially for geek girls.
Calculates sines, degrees, cosines, factorials, square roots, logarithms.
I found the source codes on the open spaces of the American-Canadian Internet.
I wrote only CSS styles myself and some HTML. JS is not mine.
All styles and scripts inside HTML are connected using tags because, according to the idea, there should be one single file.
The design is mine. The color scheme was "stretched" by me with the help of Adobe Color from the picture with the thread bobbin.

Adobe Color locates: https://color.adobe.com/create
